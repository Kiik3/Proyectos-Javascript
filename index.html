Todos los proyectos han sido desarrollados utilizando las tecnologías: HTML5, CSS3, Bootstrap y Javascript.

Gestor de Tareas

Descripción: Hace uso del LocalStorage, este proyecto permite la gestión: ingreso y eliminación de tareas (cosas por hacer).

Simulador de Carrito de Compras

Descripción: Hace uso del LocalStorage, este proyecto simula el funcionamiento de un carrito de compras, en el que se pueden agregar productos deseados, eliminar individualmente o todos a la vez.

Simulador de Envío de Email

Descripción: Este proyecto simula el envío de un email, verifica que se ingrese un correo válido y que los campos no estén vacíos.

Cotizador de Seguro de Autos

Descripción: Este proyecto permite cotizar seguros de autos (americano, europeo, asiático) a partir de parámetros establecidos previamente en el programa.

Gestiona tu Presupuesto

Descripción: Hace uso del LocalStorage, este proyecto permite gestionar el presupuesto, a partir de un monto inicial y gastos añadidos se calcula e indica el presupuesto restante.

Conversor a Criptomonedas

Descripción: Este proyecto consume una REST API, permite seleccionar la criptomoneda de la que se desea saber su equivalencia respecto a una moneda seleccionada (USD, EUR, MXN).

Localizador de Gasolineras

Descripción: Este proyecto consume una REST API, permite conocer la ubicación de las gasolineras en México, además se puede filtrar la información mediante un buscador.

Buscador de Canciones

Descripción: Este proyecto consume una REST API, permite la búsqueda de letras de canciones.

Filtro de Búsqueda de Autos

Descripción: Este proyecto permite la búsqueda de autos, mediante múltiples filtros existentes.

Reserva de Citas

Descripción: Hace uso de IndexedDB, este proyecto permite llevar un control de citas, donde se puede ingresar una cita, eliminarla y observar el listado de las citas existentes.
